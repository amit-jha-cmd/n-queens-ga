# n-queens-ga

## Installation

```shell
pip install -r requirement.txt
```


## How to run

```shell
python main.py --n 3 --epochs 1 --m 5 --history true
```

:warning: Here n is n in n-queens, Epoch is number of iterations, m is the population size
